# Roadmap — Delirium AI

**Version :** 2.0 | **Date :** 12 avril 2026

*De l'app personnelle au wiki collectif de la connaissance.*

---

## Phase 0 — Fondation (T2 2026 : Avril-Juin)

### Livrables
- [ ] Arborescence projet + documentation complète (ce dossier)
- [ ] Prototype CLI du S1 (sparring partner avec persona calibré)
- [ ] System prompts S1 + S2 validés sur corpus de test (phrase d'ancrage, MI, H, persona vector)
- [ ] Non-BlocNote : prototype de l'app déjà habitée (notes autonomes Delirium)
- [ ] Pipeline OSINT basique (APIs publiques)
- [ ] Prototype profilage inversé (10 scénarios testés)
- [ ] Base SQLite + schéma de données v1

### Critères Go/No-Go Phase 1
- Le S1 produit des réponses amicales+franches, adaptées au profil (pas sycophantes, pas toxiques)
- Le profilage inversé déclenche une correction spontanée dans > 70% des tests
- L'architecture supporte oubli sélectif + logs d'exécution obligatoires

## Phase 1 — MVP Mobile (T3 2026 : Juillet-Septembre)

### Livrables
- [ ] App mobile (Flutter) — Non-BlocNote, mono-bouton, capture vocale, sparring texte
- [ ] Whisper STT intégré (local, on-device)
- [ ] Onboarding archétype inversé fonctionnel (premier message toujours neutre-léger)
- [ ] Phase Probing → Confident Muet → Reflet → Sparring (progression automatique)
- [ ] Variable H dynamique + persona non-complémentaire (adjacence sémantique)
- [ ] Protocole danger 3 niveaux + contact ICE
- [ ] Signalement comportement IA + correction persona
- [ ] Base vectorielle locale + état épistémique {H,C+,E,B}
- [ ] Chiffrement local (SQLCipher)
- [ ] 50 bêta-testeurs (friends & family)

### Critères Go/No-Go Phase 2
- Rétention J7 > 40%
- Rétention J30 > 20%
- NPS > 30 parmi les bêta-testeurs
- Aucun incident de sécurité ou de confidentialité
- Le registre de ton détecté correspond au registre réel dans > 60% des cas

## Phase 2 — Cold Weaver + Import IA + Vie Autonome (T4 2026)

### Livrables
- [ ] Pipeline Cold Weaver v1 (ArXiv + GitHub) avec collision score SerenQA
- [ ] Import historiques ChatGPT (format JSON) + sycophancy score
- [ ] Détection d'inspirations avortées (4 critères)
- [ ] Vie autonome de Delirium dans le Non-BlocNote (notes = collisions reformulées)
- [ ] Détection de défensivité (PsyFIRE) + IPCTracker (Circumplex Interpersonnel)
- [ ] Notifications de collision
- [ ] Première version du module Vision du Monde (détection de boucles)
- [ ] Oubli sélectif opérationnel
- [ ] 500 bêta-testeurs (bêta publique limitée)

### Critères Go/No-Go Phase 3
- Time-to-first-collision < 30 jours
- Taux de clic notifications > 40%
- Faux positifs collisions < 30%
- Au moins 1 "moment eurêka" rapporté par > 10% des testeurs

## Phase 3 — Viralité + Polish (T1 2027 : Janvier-Mars)

### Livrables
- [ ] Système d'invitation (signée/anonyme) avec description custom
- [ ] Import Claude + Gemini
- [ ] Sync cloud FR optionnel (E2E chiffré)
- [ ] App desktop (Tauri)
- [ ] TTS optionnel
- [ ] AIPD (Analyse d'Impact Protection des Données) réalisée
- [ ] Audit de sécurité externe

### Critères Go/No-Go Lancement
- Coefficient viral (K) > 0.5 (chaque utilisateur invite en moyenne 0.5 personne)
- Conformité RGPD validée par DPO ou conseil juridique
- Pen test passé sans vulnérabilité critique

## Phase 6 — OmniArxiv (2028+)

*Le but ultime. L'utilisateur autonome ne part pas — il évolue.*

### Livrables
- [ ] Plateforme de publication open-source (wiki des idées)
- [ ] Comités de lecture IA + humains
- [ ] Système de peer-review où les IA créent de nouvelles connexions entre publications
- [ ] Transition utilisateur : consommateur → générateur de connaissance
- [ ] Open-source du moteur de collision et du formalisme
- [ ] Multilangue, multi-domaine, multi-culture

### Mesure de Succès
- L'utilisateur n'a plus besoin de Delirium personnel → il publie sur OmniArxiv
- Le jardin personnel est devenu un jardin collectif
- Quelque soit le domaine, **nous** sommes l'intelligence
